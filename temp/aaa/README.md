## 安装
```bash
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
source .venv/bin/activate  # Linux/macOS
.venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

# 安装开发模式
pip install -e .
```
